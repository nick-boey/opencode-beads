Get the current beads project status:

**Project Statistics:**
!`bd stats`

**Blocked Issues (need attention):**
!`bd blocked`

**Work In Progress:**
!`bd list --status=in_progress`

**Recent Activity (last 5 issues):**
!`bd list --limit=5`

Summarize:
1. Overall project health (open vs closed ratio)
2. Any blocked work that needs attention or unblocking
3. Current work in progress and who's working on what
4. Recommendations for next steps

If there are blocked issues, explain what's blocking them.
